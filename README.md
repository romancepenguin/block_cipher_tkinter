# block_cipher_tkinter

대학 과제
문자열 암호화/복호화 GUI 프로그램


## 기능
- EBC, CBC , CFB , OFB , CTR 블록 암/복호화 기능 제공
- 사전 공격(Dictionary attack) 기능 제공

## 실행 환경
- window 10
- python3.9
